**Deskripsi Data Walmart:**
-
Walmart merupakan salah satu perusahaan ritel terbesar di dunia yang berasal dari Amerika Serikat. Konsepnya seperti supermarket besar / hypermarket dengan harga murah.
Dan hingga sekarang, Walmart punya ribuan toko di banyak negara dan juga beroperasi secara online.

**Penjelasan Tugas:**
-
Tugas ini untuk melatih membuat dashboard interaktif dengan menggunakan streamlit, data yang digunakan adalah data pelanggan Walmart. Dimana terdapat kolom karakteristik pelanggan diantaranya kolom:

- usia
- jenis kelamin
- penggunaan diskon
- metode pembayaran
- kota tempat tinggal
- kategori produk
- nama produk
- jumlah pembelian
- tanggal pembelian
- id pelanggan

**Fitur Yang Digunakan:**
-
untuk visualisasi menggunakan:
- line chart: untuk mengetahui trend pelanggan dan trend penjualan di walmart
- pie chart: untuk mengetahui komposisi perbandingan setiap kategori dari jenis kelamin, kategori produk dan penggunaan diskon
- bar chart: untuk mengetahui distribusi dari rentang usia dan metode pembayaran

untuk sisi kiri dashboard:
- menggunakan fungsi dari streamlit yaitu st.sidebar()

untuk filter:
- menggunakan multiselect()
- sehingga user bisa memilih filter apa yang ingin ditampilkan
